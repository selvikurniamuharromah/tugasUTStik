RUMUS BANGUN RUANG SISI DATAR

1. RUMUS KUBUS

Rumus Volume Kubus (V) = s x s x s

Rumus Luas Permukaan Kubus (Lp) = 6 x s x s

Keterangan:

V = volume kubus

Lp = luas permukaan kubus

s = sisi kubus

2. RUMUS BALOK

Rumus Volume Balok (V) = p x l x t

Rumus Luas Permukaan Balok (Lp) = 2 x ( pl + lt + pt)

Keterangan:

V = volume balok

Lp = luas permukaan balok

p = panjang balok

l = lebar balok

t = tinggi balok

3. RUMUS PRISMA

Rumus Volume Prisma (V) = LA x t

Rumus Luas Permukaan Prisma (Lp) = (2 x LA) + (KA x t)

Keterangan:

V = volume prisma

Lp = luas permukaan prisma

LA = luas alas prisma

KA = keliling alas prisma

t = tinggi prisma

4. RUMUS LIMAS

Rumus Volume Limas (V) = 1/3 x LA x t

Rumus Luas Permukaan Limas (Lp) = LA + LS

Keterangan:

V = volume limas

Lp = luas permukaan limas

LA = luas alas limas

LS = luas selubung limas

t = tinggi limas
